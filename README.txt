Globalad Block

Globalad is a Moodle block that allows you to create the same personalized ad for all courses and users dashboard. This ad will always be the same for all courses and user dashboard where you add the Globalad block and behaves just as HTML Block.

Globalad es un bloque para Moodle que permite crear el mismo anuncio personalizado para todos los cursos y area personal de usuarios. Este anuncio siempre será el mismo en todos los cursos y area personal de usuarios en los que añadas el bloque y se comporta como un bloque HTML.


Installation

Globalad should be installed like any other block and select the type of file you want to search. See https://docs.moodle.org/36/en/Installing_plugins
